# EhMagnet

花十分钟写了一个脚本 爬Ex收藏本子的磁力链接

# 食用指南
 - 将代码中的cookie 替换成你的cookie
 
![pic.png](https://i.loli.net/2019/07/26/5d3aa7d8d8f4768680.png)
 ```
打开Ex

按F12打开开发者工具

点Network 按F5刷新后点击第一个

圈内冒号后面的一长串就是你的Cookie
```
 - 运行

 - 喝茶

 - 等待结束
